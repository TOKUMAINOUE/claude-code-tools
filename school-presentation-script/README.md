### 在学中の話：加藤さんとの出会い

【スライド】LPの「学校の授業は正直物足りなかった」のあたり

学校の授業は正直物足りなかった。課題は片手間で一瞬で終わらせて、授業中はずっと別の仕事をしてた。あんまり良くないけどね。

非常勤講師で加藤さんっていう人がいて。映像制作の現場で画面デザインをやってる人。ドラマとか映画に映るPC・スマホの画面をデザインする仕事。この人と仕事がしたかった。

でも自分、人に話しかけるのが苦手で。友達の村上に「加藤さんと仕事したいんだよね」って言ったら、村上が勝手に営業しに行ってくれた。「井上はAdobe全般使えます、3Dもできます、動画編集もできます」って。

【間】少し笑いを入れる間。

結局、自分がやってたのは加藤さんの課題解決だった。「こういう見せ方をしたいならXDをこう組んだらどうですか」みたいな。技術で相手の問題を解くこと。これが全ての始まり。

### SOELに入って変わったこと

【スライド】LPの「SOELに入って、変わった」の部分

SOELに入って、変わった。入社直後からCMSのUI開発、東京GXアクションの映像・広告制作、社内HPのデザイン〜実装。物量で戸惑った。でも楽しかった。

一番変わったのは、言語化ができるようになったこと。デザインの言語化。自分の考えの言語化。

あと、街を歩いてて目に入るデザイン、インスタのフィード、全部に頭の中でダメ出しするようになった。「自分ならもっとこうする」って。それを繰り返してたらデザイン力が上がったし、説得力のあるデザインができるようになった。

今は動画編集・デザイン・システム開発・採用・新しい商材の設計・AI開発、全部やってる。
